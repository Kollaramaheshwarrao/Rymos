# **App Name**: RymosAI

## Core Features:

- Code/MCQ Submission: Allows students to submit code snippets or answer multiple-choice questions.
- Error Analysis: The AI tool identifies the type of mistake (logic flaw, concept misunderstanding, syntax error, etc.) in the submitted code or MCQ answer.
- Step-by-Step Explanation: Generates a detailed, step-by-step explanation of why the student's answer is incorrect and provides the correct approach.
- Code Execution: Optionally executes code snippets in a secure sandbox (Judge0/Piston) to detect runtime or syntax errors.
- AI Feedback Display: Displays AI-generated feedback, including the error type, explanation, correct solution, and a pro tip, in a card-style format with tabs.
- Progress Tracking: Tracks student progress and identifies recurring mistakes to suggest personalized exercises.
- Sandboxed Code: Integration with Judge0/Piston to securely execute code snippets.

## Style Guidelines:

- Primary color: Deep blue (#3498db) for a modern and intellectual feel.
- Background color: Light gray (#ecf0f1) to provide a clean and unobtrusive backdrop.
- Accent color: Vibrant orange (#e67e22) to highlight key interactive elements.
- Headline font: 'Space Grotesk' sans-serif, used for headers, conveying a technical but approachable aesthetic; Body font: 'Inter' sans-serif for readability and a modern machined look.
- Code font: 'Source Code Pro' monospace font for code snippets.
- Use a set of crisp, clear icons (Feather or FontAwesome) to represent different error types and actions.
- Subtle animations (e.g., fade-in, slide-up) for AI feedback and UI transitions to enhance user experience.
